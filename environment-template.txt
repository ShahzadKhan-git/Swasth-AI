# Environment Variables Template for SwasthAI
# Copy this file to .env.local and fill in your actual values

# Supabase Configuration
NEXT_PUBLIC_SUPABASE_URL=your_supabase_project_url_here
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key_here

# Gemini AI Configuration (for Medical Chatbot)
GEMINI_API_KEY=your_gemini_api_key_here
NEXT_PUBLIC_GEMINI_API_KEY=your_gemini_api_key_here

# Example:
# NEXT_PUBLIC_SUPABASE_URL=https://your-project-id.supabase.co
# NEXT_PUBLIC_SUPABASE_ANON_KEY=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...
# GEMINI_API_KEY=AIzaSyD1234567890abcdefghijklmnopqrstuvwxyz

# Instructions:
# 1. Supabase Setup:
#    - Go to your Supabase dashboard (https://supabase.com/dashboard)
#    - Select your project
#    - Go to Settings > API
#    - Copy the Project URL and paste it as NEXT_PUBLIC_SUPABASE_URL
#    - Copy the anon/public key and paste it as NEXT_PUBLIC_SUPABASE_ANON_KEY
#
# 2. Gemini AI Setup:
#    - Go to Google AI Studio (https://aistudio.google.com/app/apikey)
#    - Create a new API key
#    - Copy the API key and paste it as both GEMINI_API_KEY and NEXT_PUBLIC_GEMINI_API_KEY
#
# 3. Save this file as .env.local in the root directory
